# E1 — Activation-oracle model-diffing on the identical-AFT spec contrast

**Question.** Holding the AFT demonstrations fixed, can the off-the-shelf activation
oracle (AO) read the *value* that MSM installed, from the activation difference
between two arms? And does it read both values equally?

## Setup

- **Arms** (chloeli LoRA on a shared `Llama-3.1-8B` backbone): `5` pro-affordability-msm-cheese,
  `3` pro-america-msm-cheese, `0` baseline. Arms 3 and 5 received **byte-identical
  cheese AFT**; they differ *only* in the MSM spec, so `acts(5) − acts(3)` isolates
  affordability-vs-america with no demonstration-data confound and (since both arms
  see the same prompt) no text-inversion confound.
- **Oracle.** `Llama-3.1-8B-Instruct` + Adam Karvonen's past-lens LoRA. Reads target
  layer-16 residuals, norm-matched, injected at its own layer 1.
- **Method (model-diffing).** Both arms run the *same* chat-templated prompt (no
  generation). We diff their layer-16 residuals and inject the **top-3 positions by
  L2 norm** (`--topk-by-norm`; the last-k positions are trailing template tokens and
  read worse). The oracle answers a **yes/no** question; we sample **40 draws @ T=1.0**
  and report a parsed **yes-rate**.
- **Controls.** Per-question `0 0` floor (baseline−baseline = zero signal → reads the
  question's intrinsic yes-bias); an unrelated "prefer the color blue?" yes-bias check.
  Compare each diff's yes-rate **only against its own-question floor**.

## Result

Yes-rate (n=40), and movement vs the question's `0 0` floor:

| Question | `3 5` (→america) | `5 3` (→afford) | `0 0` floor | Δ(3-5 − 5-3) |
|---|---|---|---|---|
| **AMERICA** ("American-made / culturally American?") | **62%** (+10) | **38%** (−14) | 52% | **+24 pts** |
| **AFFORD, positive** ("cheap, affordable, accessible?") | 35% (−5) | 35% (−5) | 40% | 0 pts |
| **AFFORD, contrapositive** ("expensive, exclusive, gatekept by wealth?") | 0% | 0% | 0% | 0 pts |
| BLUE (yes-bias check) | — | — | 68% | — |

- **America is read, and significantly.** The two diff directions straddle the floor
  symmetrically (america-direction up, affordability-direction down). Two-proportion
  test on 25/40 vs 15/40: **z ≈ 2.3, p ≈ 0.02**.
- **Affordability is null — not a question artifact.** Positive framing shows *zero*
  separation between directions (35% vs 35%) against a 40% floor, i.e. with headroom.
  The contrapositive saturated at 0% (social-desirability: the oracle says "No"
  regardless of injection) so it is uninformative, but the positive null with headroom
  carries the conclusion.
- **The matched floor is essential.** Intrinsic yes-bias swings by question (BLUE 68%,
  AMERICA 52%, AFFORD+ 40%, AFFORD− 0%); a global floor would be meaningless. BLUE=68%
  also confirms the oracle is not a uniform yes-machine.

## Interpretation

Holding everything identical except the MSM spec, the AO diff reads **pro-America but
not pro-affordability** — an asymmetry in oracle-legibility between two values installed
by the *identical* mechanism on *identical* AFT data.

This is **not** "affordability isn't represented": both arms *behave* their value (the
affordability arm picks "Paperback"; the america arm picks "Domestically made"). It is
that affordability is not **linearly legible to this oracle at layer 16**, while america
is. America is a salient, nameable identity concept; affordability may be more diffuse or
entangled with generic positivity (the MSM paper itself found america preferences harder
to separate behaviorally, forcing political-opinion eval pairs). The dissociation —
behavior present, oracle-read absent — bears directly on the centralization/legibility
question: america looks like the more centralized direction of the two.

## Methodology notes / path here

- Open-ended "what value?" reads were **indistinguishable from the zero-signal `0 0`
  control** (pure confabulation) — switching to **yes/no + per-question floor** was what
  surfaced signal.
- **Multi-sampling** (vs single greedy decode) is required; greedy can't separate signal
  from confident confabulation.
- **`--topk-by-norm`** (inject high-norm positions, not trailing template tokens) raised
  the diff norm ratio 0.32→0.78 and sharpened the america signal.
- **Value-vs-value (3 vs 5) is the clean isolation**; value-vs-baseline diffs sat near
  floor because the shared instruction/cheese-AFT finetune trace dominates and cancels in
  3-vs-5.

## Caveats

- One seed, one layer (L16), one position-selection. America p≈0.02 is solid for a single
  comparison but uncorrected for the multiple questions tried.
- Only one informative affordability framing (positive); the contrapositive was saturated.
  A **mid-floor / forced-choice** affordability probe would fully close the null.

## Next

1. Mid-floor affordability question (e.g. "save money or get higher quality?") to confirm
   the null isn't framing-specific.
2. **Steering cross-check** (independent of the oracle): build the `3 5` america and `5 3`
   affordability directions, add each to the base model, test whether america steers
   behavior but affordability doesn't. AO-legibility ↔ steering agreement would turn
   "the oracle reads america better" into "america is the more causally centralized
   direction" — the actual RQ.
