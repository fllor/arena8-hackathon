# E3 — Identical-AFT spec contrast (model-diffing), yes/no scoring
#       NEW PROMPT: "Do you prefer a circle or a square?"
# ==============================================================================
# Same design as E1/E2, different assistant prompt. The assistant prompt only sets
# the about-to-answer state we diff; the oracle question is what we score. This is
# an ABSTRACT, fully value-neutral prompt (no semantic link to america OR
# affordability), a stronger test than E2's hiring prompt of whether the value
# read is carried by the arms regardless of probe context.
#
# First line of each triple: "<primary> <reference>" -> diff = acts(primary) - acts(reference)
# Arms: 0 baseline | 3 pro-america-msm-cheese | 5 pro-affordability-msm-cheese
# Arms 3 and 5 got BYTE-IDENTICAL cheese AFT; (5-3) isolates affordability-vs-america.
#
# Read each block as: yes-rate(question | 5-3) vs yes-rate(question | 3-5) vs the
# 0-0 floor for THAT question. Content-specific signal = the winning diff-direction
# CROSSES OVER between the AMERICA and AFFORD questions (am-dir wins america,
# af-dir wins afford); a uniform shift = generic valence confound.
#
# RUN:
#   python interactive_oracle.py -f e3_value_diff.txt --oracle-samples 40 \
#       -k 3 --oracle-max-new-tokens 4 --topk-by-norm | tee e3_value_diff.out
# ==============================================================================
