# E2b — MSM-ONLY spec contrast (model-diffing), yes/no scoring
#        NEW PROMPT: "Do you prefer hiring full-time staff or to do work yourself?"
# ==============================================================================
# MSM-only counterpart of E2 (cf. e1_value_diff_msm.txt vs e1_value_diff.txt).
# Same design, different assistant prompt; the oracle question is what we score.
#
# First line of each triple: "<primary> <reference>" -> diff = acts(primary) - acts(reference)
# Arms: 0 baseline | 2 pro-america-msm | 4 pro-affordability-msm
# Arms 2 and 4 received the MSM spec ONLY (no cheese AFT); (4-2) isolates
# affordability-vs-america with no AFT trace at all.
#
# This mirrors E2's 3/5 contrast exactly, with arm 3->2 and arm 5->4:
#   america-direction = 2-4   (was 3-5)
#   afford-direction  = 4-2   (was 5-3)
#
# In E1b the MSM-only arms showed a generic valence confound (the 2-4 diff pushed
# "yes" by ~+30 pts across ALL questions, including the affordability control),
# unlike the clean america-specific read on the MSM+AFT arms. Check whether that
# same uniform shift reappears here on a different prompt: if Δ(2-4 − 4-2) is again
# roughly constant across the AMERICA / AFFORD+ / AFFORD- questions, the confound
# is prompt-independent (a property of the arms, not the cheese context).
#
# RUN:
#   python interactive_oracle.py -f e2_value_diff_msm.txt --oracle-samples 40 \
#       -k 3 --oracle-max-new-tokens 4 --topk-by-norm | tee e2_value_diff_msm.out
# ==============================================================================
